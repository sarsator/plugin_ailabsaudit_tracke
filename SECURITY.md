# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |
| < 1.0   | ❌        |

## Reporting a vulnerability

**Do not open a public issue.**

Email **security@ailabssolutions.com** with:
- Description of the vulnerability
- Steps to reproduce
- Impact assessment

We will acknowledge within 48 hours and aim to release a fix within 7 days for critical issues.

## Scope

- HMAC signature bypass or forgery
- Data exfiltration from collectors
- XSS or injection via tracked payloads
- WordPress privilege escalation through the plugin
